# anycard

iOS app for adding custom loyalty cards to Apple Wallet.

## Features

- Add any loyalty card to Apple Wallet
- Scan existing cards with camera (barcode/QR)
- Manual entry (name + code)
- Custom graphics (user-uploaded images)

## Tech Stack

- Swift + SwiftUI
- PassKit (Apple Wallet integration)
- AVFoundation (camera scanning)
- CoreImage (barcode/QR generation)
- SwiftData (local storage)
